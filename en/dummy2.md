#Formatting Markup

##Ordered list
When listing steps in a guide it should look like this:
1. Make my changes
  1. Fix bug
  2. Improve formatting
    * Make the headings bigger
2. Push my commits to GitHub
3. Open a pull request
  * Describe my changes
  * Mention all the members of my team
    * Ask for feedback

**1. One**
2. Two
3. Three


##Bullet lists
- Make my changes  
   - Fix bug
   - Improve formatting
      - Make the headings bigger
- Push my commits to GitHub
   - Open a pull request  
      - Describe my changes
- Mention all the members of my team
   - Ask for feedback
